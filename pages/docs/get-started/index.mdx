# Gemini CLI 문서

Gemini CLI는 Gemini의 강력한 기능을 터미널로 직접 가져오는 오픈 소스 AI 에이전트입니다.
개발자, 엔지니어, SRE 등을 위한 터미널 우선의 확장 가능하고 강력한 도구로 설계되었습니다.

Gemini CLI는 로컬 환경과 통합됩니다. 프로젝트 컨텍스트를 유지하면서 파일을 읽고 편집하고,
셸 명령을 실행하고, 웹을 검색할 수 있습니다.

## 시작하기

환경을 설정하고 기본 사항을 학습하여 Gemini CLI와 함께하는 여정을 시작하세요.

- **[빠른 시작](/docs/get-started/quick-start):** 몇 분 만에 채팅을 시작할 수 있는 간소화된 가이드입니다.
- **[설치](/docs/get-started/installation):** macOS, Linux 및 Windows용 설치 지침입니다.
- **[인증](/docs/get-started/authentication):** Google OAuth, API 키 또는 Vertex AI를 사용하여 액세스를 설정합니다.
- **[예제](/docs/get-started/examples):** 일반적인 사용 시나리오를 보고 워크플로에 대한 영감을 얻으세요.

## Gemini CLI 사용하기

Gemini CLI가 시스템과 안전하고 효과적으로 상호 작용할 수 있도록 하는 핵심 기능을 마스터하세요.

- **[CLI 사용법](/docs/get-started/cli-reference):** 명령줄 인터페이스의 기본 사항을 알아봅니다.
- **[파일 관리](/docs/use-gemini-cli/file-management):** 모델에 코드를 읽고 파일에 직접 변경 사항을 적용할 수 있는 권한을 부여합니다.
- **[셸 명령](/docs/use-gemini-cli/shell-commands):** 모델이 빌드, 테스트 및 git 명령을 실행하도록 허용합니다.
- **[메모리 관리](/docs/use-gemini-cli/memory-management):** 세션 간에 지속되는 프로젝트 및 기본 설정에 대한 사실을 Gemini CLI에 가르칩니다.
- **[프로젝트 컨텍스트](/docs/use-gemini-cli/project-context):** `GEMINI.md` 파일을 사용하여 프로젝트에 영구적인 컨텍스트를 제공합니다.
- **[웹 검색 및 가져오기](/docs/use-gemini-cli/web-search-and-fetch):** 모델이 인터넷에서 실시간 정보를 가져오도록 활성화합니다.
- **[세션 관리](/docs/use-gemini-cli/session-management):** 채팅 세션을 저장, 재개 및 정리합니다.

## 설정

워크플로와 기본 설정에 맞게 Gemini CLI를 사용자 지정하세요.

- **[설정](/docs/configuration/setting):** 응답 창의성, 출력 상세도 등을 제어합니다.
- **[모델 선택](/docs/configuration/model-selection):** 특정 작업에 가장 적합한 Gemini 모델을 선택합니다.
- **[무시 파일](/docs/configuration):** `.geminiignore`를 사용하여 민감한 파일을 모델의 컨텍스트에서 제외합니다.
- **[신뢰할 수 있는 폴더](/docs/configuration/trusted-folders):** 파일 액세스 및 실행에 대한 보안 경계를 정의합니다.
- **[토큰 캐싱](/docs/configuration/token-caching):** 컨텍스트를 캐싱하여 성능과 비용을 최적화합니다.
- **[테마](/docs/configuration/themes):** CLI의 시각적 모양을 개인화합니다.

## 고급 기능

복잡한 워크플로와 엔터프라이즈 환경을 위한 강력한 기능을 살펴보세요.

- **[헤드리스 모드](/docs/advanced-features/headless-mode-and-scripting):** 자동화된 추론을 위해 스크립트 또는 CI/CD 파이프라인에서 Gemini CLI를 실행합니다.
- **[샌드박싱](/docs/advanced-features/sandboxing):** 안전한 격리된 컨테이너에서 신뢰할 수 없는 코드나 도구를 실행합니다.
- **[체크포인팅](/docs/advanced-features):** 작업 공간 상태를 저장하고 복원하여 실험적 변경 사항에서 복구합니다.
- **[사용자 정의 명령](/docs/advanced-features/custom-commands):** 자주 사용하는 프롬프트에 대한 단축키를 만듭니다.
- **[시스템 프롬프트 재정의](/docs/advanced-features/system-prompt-override):** 모델에 제공되는 핵심 지침을 사용자 지정합니다.
- **[텔레메트리](/docs/advanced-features/telemetry):** 사용 데이터가 수집되고 관리되는 방식을 이해합니다.
- **[엔터프라이즈](/docs/advanced-features/enterprise-features):** 대규모 팀을 위한 구성 및 정책을 관리합니다.

- **[확장 프로그램 참조](/docs/extensions/extensions-reference):** 확장 형식, 명령 및 구성을 깊이 이해합니다.
- **[모범 사례](/docs/extensions/best-practices):** 훌륭한 확장 프로그램을 구축하기 위한 전략을 배웁니다.

- **[후크](/docs/ecosystem-and-extensibility/hooks):** 특정 CLI 이벤트에서 실행되는 스크립트를 작성합니다.
- **[에이전트 기술](/docs/ecosystem-and-extensibility/creating-agent-skills):** 전문 지식과 워크플로를 추가합니다.
