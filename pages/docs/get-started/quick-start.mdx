# Gemini CLI 시작하기

Gemini CLI에 오신 것을 환영합니다! 이 가이드는 Gemini CLI를 설치, 구성 및 사용하여 터미널에서 직접 워크플로를 향상시키는 데 도움을 줍니다.

## 빠른 시작: Gemini CLI 설치, 인증, 구성 및 사용

Gemini CLI는 고급 언어 모델의 기능을 명령줄 인터페이스로 직접 가져옵니다. AI 기반 어시스턴트인 Gemini CLI는 코드 이해 및 생성부터 문서 검토 및 편집에 이르기까지 다양한 작업을 지원할 수 있습니다.

## 설치

Gemini CLI를 설치하고 실행하는 표준 방법은 `npm`을 사용합니다.

```bash
npm install -g @google/gemini-cli
```

Gemini CLI가 설치되면 명령줄에서 Gemini CLI를 실행합니다.

```bash
gemini
```

더 많은 설치 옵션은 [Gemini CLI 설치](/docs/get-started/installation)를 참조하세요.

## 인증

Gemini CLI를 사용하기 시작하려면 Google 서비스로 인증해야 합니다. 대부분의 경우 기존 Google 계정으로 로그인할 수 있습니다.

1. 설치 후 Gemini CLI를 실행합니다.

   ```bash
   gemini
   ```

2. "How would you like to authenticate for this project?"라는 질문이 나오면 **1. Login with Google**을 선택합니다.

3. Google 계정을 선택합니다.

4. **Sign in**을 클릭합니다.

특정 계정 유형의 경우 Google Cloud 프로젝트를 구성해야 할 수 있습니다. 다른 인증 방법을 포함한 자세한 내용은 [Gemini CLI 인증 설정](/docs/get-started/authentication)을 참조하세요.

## 구성

Gemini CLI는 환경 변수, 명령줄 인수 및 설정 파일을 포함하여 동작을 구성하는 여러 가지 방법을 제공합니다.

구성 옵션을 살펴보려면 [Gemini CLI 구성](/docs/get-started/configuration)을 참조하세요.

## 사용

설치 및 인증이 완료되면 터미널에서 명령과 프롬프트를 실행하여 Gemini CLI를 사용할 수 있습니다. 코드 생성, 파일 설명 등을 요청하세요.

Gemini CLI의 강력한 기능을 살펴보려면 [Gemini CLI 예제](/docs/get-started/examples)를 참조하세요.
