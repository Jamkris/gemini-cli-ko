# Gemini CLI: 라이선스, 서비스 약관 및 개인정보 처리방침

Gemini CLI는 명령줄 인터페이스에서 직접 Google의 강력한 AI 서비스와 상호 작용할 수 있는
오픈 소스 도구입니다. Gemini CLI 소프트웨어는
[Apache 2.0 라이선스](https://github.com/google-gemini/gemini-cli/blob/main/LICENSE)에 따라
라이선스가 부여됩니다. Gemini CLI를 사용하여 Google 서비스에 액세스하거나 사용하는 경우,
해당 서비스에 적용되는 서비스 약관 및 개인정보 처리방침이 해당 액세스 및 사용에 적용됩니다.

Gemini CLI 사용 통계는 Google의 개인정보 처리방침에 따라 처리됩니다.

**참고:** Gemini CLI 사용에 적용되는 할당량 및 가격 세부 정보는
[할당량 및 가격](/docs/resources/quota-and-pricing)을 참조하세요.

## 지원되는 인증 방법

인증 방법은 Gemini CLI로 Google 로그인하고 서비스에 액세스하는 데 사용하는 방법을 말합니다.
지원되는 인증 방법은 다음과 같습니다:

- Google 계정으로 Gemini Code Assist에 로그인.
- Gemini Developer API에서 API 키 사용.
- Vertex AI GenAI API에서 API 키 사용.

앞서 언급한 Google 서비스에 적용되는 서비스 약관 및 개인정보 처리방침은 아래 표에 나와 있습니다.

Google 계정으로 로그인했는데 Google 계정과 연결된 Gemini Code Assist 계정이 아직 없는 경우,
개인용 Gemini Code Assist 가입 절차로 안내됩니다. Google 계정이
조직에서 관리되는 경우, 관리자가 개인용 Gemini Code Assist에 대한 액세스를 허용하지 않을 수 있습니다.
자세한 내용은 [개인용 Gemini Code Assist FAQ](https://developers.google.com/gemini-code-assist/resources/faqs)를
참조하세요.

사용자의 경우 다음 서비스 약관 및 개인정보 처리방침 문서가 적용됩니다:

- 개인용 Gemini Code Assist:
  [Google 서비스 약관](https://policies.google.com/terms) 및
  [개인용 Gemini Code Assist 개인정보 처리방침](https://developers.google.com/gemini-code-assist/resources/privacy-notice-gemini-code-assist-individuals).
- Google AI Pro 또는 Ultra 구독이 포함된 Gemini Code Assist:
  [Google 서비스 약관](https://policies.google.com/terms),
  [Google One 추가 서비스 약관](https://one.google.com/terms-of-service)
  및 [Google 개인정보 처리방침\*](https://policies.google.com/privacy).
- Gemini Code Assist Standard 및 Enterprise 에디션:
  [Google Cloud Platform 서비스 약관](https://cloud.google.com/terms) 및
  [Google Cloud 개인정보 처리방침](https://cloud.google.com/terms/cloud-privacy-notice).

_\* 귀하의 계정이 Gemini Code Assist Standard 또는 Enterprise 에디션의 활성 구독과도
연결되어 있는 경우, Gemini Code Assist의 모든 사용에 대해 Gemini Code Assist Standard 또는
Enterprise 에디션의 약관 및 개인정보 처리방침이 적용됩니다._
