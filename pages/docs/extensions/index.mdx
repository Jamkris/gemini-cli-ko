# Gemini CLI 확장

Gemini CLI 확장은 프롬프트, MCP 서버, 커스텀 명령, 훅, 서브 에이전트 및 에이전트 기술을 친숙하고 사용자 친화적인 형식으로 패키징합니다. 확장을 사용하면 Gemini CLI의 기능을 확장하고 그 기능을 다른 사람들과 공유할 수 있습니다. 확장은 쉽게 설치하고 공유할 수 있도록 설계되었습니다.

확장의 예시를 보려면 [Gemini CLI 확장](https://geminicli.com/extensions/browse/) 갤러리를 찾아볼 수 있습니다.

## 확장 관리

대화형 명령을 사용하여 설치된 확장과 해당 상태를 확인할 수 있습니다:

```bash
/extensions list
```

또는 비대화형 모드에서:

```bash
gemini extensions list
```

## 설치

실제 확장을 설치하려면, 비대화형 모드에서 GitHub 저장소 URL과 함께 `extensions install` 명령을 사용할 수 있습니다. 예를 들어:

```bash
gemini extensions install https://github.com/gemini-cli-extensions/workspace
```
